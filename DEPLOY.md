# 🐍 Деплой на PythonAnywhere — пошагово

PythonAnywhere — самый простой бесплатный хостинг для Python-бота.
Никаких Docker, GitHub, Render — всё через веб-интерфейс.

## Шаг 1: Зарегистрируйся

1. Зайди на https://www.pythonanywhere.com
2. Нажми «Pricing & signup» → выбери **Beginner** (бесплатно навсегда)
3. Подтверди email

## Шаг 2: Загрузи бота

1. В верхнем меню выбери **Files**
2. Нажми **Upload a file** и загрузи zip-архив с ботом
3. Открой **Bash console** (вкладка **Consoles** → New console → Bash)
4. Распакуй архив:
   ```bash
   unzip oge_telegram_bot_v7.zip
   cd oge_telegram_bot_v7
   ls
   ```
   Должны быть `bot.py`, `config.py`, `subjects/` и т. д.

## Шаг 3: Установи зависимости

В той же Bash-консоли:
```bash
pip install --user python-telegram-bot==20.7
```
Установка занимает ~30 секунд. Игнорируй жёлтые предупреждения.

## Шаг 4: Запусти бота

В Bash-консоли:
```bash
python3 bot.py
```

Если всё нормально, увидишь сообщение типа:
```
🤖 Бот запущен! Нажми Ctrl+C для остановки.
```

Иди в Telegram → твой бот → жми **/start**.

## Шаг 5: Чтобы бот работал, когда консоль закрыта

На бесплатном тарифе PythonAnywhere есть нюанс: **процесс умирает,
когда ты выходишь из консоли**. Решения:

### Вариант А: Always-on task (платно — $5/мес)
Самое простое, но не бесплатно.

### Вариант Б: Бесплатно — закрывай вкладку, не консоль
Достаточно НЕ закрывать вкладку Bash-консоли в браузере.
Можно свернуть, можно усыпить компьютер — бот работает,
пока вкладка PythonAnywhere открыта в браузере.
Сессия живёт несколько часов.

### Вариант В: Hacks для всегда-онлайн
1. Создай **scheduled task** (вкладка **Tasks**)
2. Запланируй ежедневный запуск `python3 /home/ТВОЙ_ЛОГИН/oge_telegram_bot_v7/bot.py`
3. Бот будет автоматически перезапускаться каждые 24 часа.

## Если что-то не работает

| Ошибка | Решение |
|--------|---------|
| `ModuleNotFoundError: telegram` | `pip install --user python-telegram-bot==20.7` |
| `Conflict: terminated by other getUpdates` | Бот уже запущен где-то ещё. Останови все копии |
| `Unauthorized` / `Invalid token` | Токен неправильный — проверь `config.py` |
| Бот не отвечает | Возможно, консоль закрылась. Открой заново и `python3 bot.py` |
