## 🚀 Как запустить

### Локально

1. Установи Python 3.10+
2. `pip install -r requirements.txt`
3. `python bot.py`
4. Открой бота в Telegram → `/start`

### На хостинге (PythonAnywhere)

См. **DEPLOY.md** — пошаговая инструкция.

## 🔧 Что под капотом

- Каждый предмет — отдельный модуль `subjects/<name>.py`
- Структура одинакова: `TASKS`, `CONSPECTS`, `DEFINITIONS`
- API: `is_ready`, `get_task_numbers`, `get_random_task`, `check_answer`,
  `get_conspect_topics`, `get_conspect`, `get_definition_categories`,
  `get_definitions`, `search_definitions`
- Чтобы добавить новый предмет — скопируй любой готовый и заполни данные

## 📚 Источники проверки

- ФИПИ-2026: демоверсии, кодификаторы, спецификации
